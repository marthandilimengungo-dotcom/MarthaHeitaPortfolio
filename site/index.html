About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, and review project history on any device — even in low-connectivity field conditions.

React Native Expo Firebase Firestore
My Contribution

My Contribution

I focused on the quality of the user experience during everyday app use — specifically the validation feedback, loading indicators, and error states. I implemented validators.js to enforce consistent input rules across forms, ErrorState.js to give users clear, friendly messages when something goes wrong, and LoadingState.js to provide visual feedback during data fetches.

My Reflection

Good error states and loading indicators seem like small details, but they define whether a user trusts an app. I studied what SiteSpy's actual failure scenarios were — network loss on a construction site, slow Firestore reads, invalid input — and designed states that address each case.
